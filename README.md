# 恐龙进化 (Dino Evolution)

一个基于 Three.js 的 3D 网页游戏，玩家操控恐龙通过吞噬比自己小的物体来成长和进化。

## 🎮 游戏介绍

在这个游戏中，你将扮演一只小恐龙，在 3D 场景中自由探索。通过触碰比自己小的物体来获取经验值，使你的恐龙体型不断成长。当经验值达到一定阈值时，恐龙会进化成更强大的品种！

### 核心玩法
- 🦖 操控恐龙在3D场景中移动 (WASD)
- 💥 触碰小物体产生爆炸并获得经验值
- 📈 体型随经验值实时增长
- 🔄 达到阈值后进化成新品种恐龙
- 🎯 从迅猛龙进化到终极棘龙

## 🛠️ 技术栈

- **3D引擎**: Three.js
- **动画**: Tween.js
- **构建工具**: Vite
- **语言**: JavaScript (ES6+)
