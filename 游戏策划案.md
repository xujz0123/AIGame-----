# 恐龙进化 - 游戏策划案

## 一、游戏概述

### 1.1 游戏名称
**恐龙进化** (Dino Evolution)

### 1.2 游戏类型
- 3D动作冒险
- 成长进化类
- 休闲竞技

### 1.3 游戏平台
- Web浏览器（桌面端和移动端）
- 基于WebGL技术

### 1.4 核心玩法
玩家操控一只恐龙在3D场景中自由移动，通过触碰比自己小的物体来获取经验值，使恐龙体型成长，达到一定经验值后进化成新品种恐龙。

## 二、游戏机制

### 2.1 角色系统

#### 2.1.1 初始角色
- **品种**: 小型恐龙（可选：迅猛龙、美颌龙等）
- **初始体型**: 基础大小（scale = 1.0）
- **初始经验值**: 0

#### 2.1.2 恐龙进化链
```
迅猛龙 (0-100经验) 
  ↓
三角龙 (100-300经验)
  ↓
剑龙 (300-600经验)
  ↓
霸王龙 (600-1000经验)
  ↓
棘龙 (1000+经验)
```

每次进化后：
- 恢复基础体型大小
- 继承累计经验值
- 解锁新的外观和能力

### 2.2 移动控制

#### 2.2.1 PC端操作
- **W**: 向前移动
- **S**: 向后移动
- **A**: 向左移动
- **D**: 向右移动
- **鼠标**: 控制视角/镜头旋转
- **空格**: 跳跃（可选）
- **Shift**: 加速冲刺（可选）

#### 2.2.2 移动端操作
- 虚拟摇杆控制移动方向
- 滑动屏幕控制视角

#### 2.2.3 移动参数
- 基础移动速度: 5 units/s
- 转向速度: 180°/s
- 加速倍率: 1.5x（可选功能）

### 2.3 碰撞与攻击系统

#### 2.3.1 碰撞检测
- 使用恐龙**头部**作为碰撞判定点
- 碰撞半径: 恐龙头部周围 0.5 units

#### 2.3.2 攻击判定
**可攻击条件**:
```javascript
物体可被攻击 = (物体体积 < 恐龙当前体积 * 0.8)
```

#### 2.3.3 攻击效果
1. **视觉效果**:
   - 物体产生爆炸特效（粒子系统）
   - 物体消失
   - 恐龙短暂发光（获得经验提示）

2. **音效**:
   - 爆炸音效
   - 吞噬/咬合音效
   - 经验获取音效

3. **数值变化**:
   - 根据物体大小获得对应经验值
   - 恐龙体型实时增长

### 2.4 成长系统

#### 2.4.1 经验值计算
```javascript
获得经验值 = 物体体积 * 10
// 例如: 体积为1的物体 = 10经验值
```

#### 2.4.2 体型成长
```javascript
当前体型 = 基础体型 * (1 + 当前阶段经验值 / 进化所需经验值)
// 例如: 在0-100经验阶段，获得50经验
// 体型 = 1.0 * (1 + 50/100) = 1.5倍
```

**体型成长上限**: 进化前最大可达基础体型的2倍

#### 2.4.3 进化阈值
| 进化等级 | 所需累计经验 | 恐龙品种 | 特点 |
|---------|------------|---------|------|
| 1 | 0 | 迅猛龙 | 灵活快速 |
| 2 | 100 | 三角龙 | 防御力强 |
| 3 | 300 | 剑龙 | 攻击范围大 |
| 4 | 600 | 霸王龙 | 强大攻击力 |
| 5 | 1000 | 棘龙 | 终极形态 |

### 2.5 场景物体系统

#### 2.5.1 物体分类
**植物类**:
- 小草 (体积: 0.5)
- 灌木 (体积: 1.0)
- 小树 (体积: 2.0)
- 大树 (体积: 5.0)

**建筑类**:
- 木桩 (体积: 1.5)
- 木屋 (体积: 3.0)
- 石屋 (体积: 6.0)
- 城堡 (体积: 10.0)

**装饰类**:
- 石头 (体积: 0.8)
- 栅栏 (体积: 1.2)
- 雕像 (体积: 4.0)

#### 2.5.2 物体分布
- 随机生成在场景中
- 根据恐龙进化等级动态调整物体分布密度
- 保证始终有可攻击的物体存在

#### 2.5.3 物体重生机制
- 被摧毁的物体在30秒后在随机位置重生
- 重生位置距离玩家至少20 units

## 四、技术实现

### 4.1 推荐技术栈

#### 4.1.1 3D引擎
- **Three.js** (推荐)
  - 轻量级，文档丰富
  - 社区活跃，资源多
  - 适合Web游戏开发

#### 4.1.2 物理引擎
- **Cannon.js** 或 **Ammo.js**
  - 碰撞检测
  - 刚体物理模拟

#### 4.1.3 模型资源
- **格式**: GLTF/GLB
- **来源**: 
  - Sketchfab (免费恐龙模型)
  - Mixamo (角色动画)
  - 自制简化模型

#### 4.1.4 其他库
- **Tween.js**: 平滑动画过渡
- **Howler.js**: 音效管理
- **Stats.js**: 性能监控

### 4.2 核心模块设计

```javascript
// 主要模块结构
├── SceneManager      // 场景管理
├── PlayerController  // 玩家控制
├── CollisionSystem   // 碰撞检测
├── ExperienceSystem  // 经验系统
├── EvolutionSystem   // 进化系统
├── ObjectSpawner     // 物体生成器
├── UIManager         // UI管理
└── ParticleSystem    // 特效系统
```

### 4.3 关键算法

#### 4.3.1 体型比较算法
```javascript
function canAttack(dino, object) {
  const dinoVolume = dino.scale.x * dino.scale.y * dino.scale.z;
  const objectVolume = object.baseVolume * object.scale.x;
  return objectVolume < dinoVolume * 0.8;
}
```

#### 4.3.2 经验成长算法
```javascript
function updateScale(dino, experience) {
  const progress = experience / dino.nextEvolutionThreshold;
  const growthFactor = 1 + Math.min(progress, 1.0);
  dino.scale.setScalar(dino.baseScale * growthFactor);
}
```

## 五、游戏流程

### 5.1 开始流程
1. 显示游戏标题界面
2. 点击"开始游戏"按钮
3. 显示操作说明（可跳过）
4. 进入游戏场景

### 5.2 游戏循环
```
1. 玩家控制恐龙移动
   ↓
2. 检测与物体碰撞
   ↓
3. 判断能否攻击
   ↓
4. 获得经验，体型增长
   ↓
5. 达到阈值触发进化
   ↓
6. 返回步骤1
```

### 5.3 结束条件（可选）
- **时间模式**: 限定时间内获得最高分
- **目标模式**: 进化到最终形态
- **无尽模式**: 无限制自由游玩
