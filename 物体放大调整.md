# 物体放大10倍调整说明

## ✅ 已完成的调整（2026-01-24）

### 📦 场景物体大小增加

所有场景物体尺寸**增加约10倍**：

| 物体类型 | 原尺寸 | 新尺寸 | 倍数 | 体积 |
|---------|--------|--------|------|------|
| 小草 | 0.3×1 | **3×10** | 10x | 0.3 |
| 灌木 | 半径0.4 | **半径4** | 10x | 0.5 |
| 木桩 | 0.6³ | **6³** | 10x | 0.7 |
| 小树 | 0.3×1.5 | **3×15** | 10x | 0.9 |
| 石头 | 1³ | **10³** | 10x | 1.2 |
| 栅栏 | 半径0.6 | **半径6** | 10x | 1.5 |
| 大树 | 0.4×2 | **4×20** | 10x | 1.8 |
| 轮胎 | 0.5环 | **5环** | 10x | 2.0 |
| 水晶 | 半径0.6 | **半径6** | 10x | 2.2 |
| 宝石 | 半径0.7 | **半径7** | 10x | 2.5 |

### 🔧 如果需要进一步调整

#### 物体太大了
编辑 `src/main.js`，减小几何体参数：
```javascript
// 例如改为8倍
new THREE.ConeGeometry(2.4, 8, 8)  // 原来是3, 10
```

#### 物体太小了
```javascript
// 例如改为12倍
new THREE.ConeGeometry(3.6, 12, 8)  // 更大
```

#### 物体太密集
```javascript
// 扩大生成范围
const distance = 20 + Math.random() * 80; // 20-100单位
```

#### 物体太分散
```javascript
// 缩小生成范围
const distance = 10 + Math.random() * 50; // 10-60单位
```
