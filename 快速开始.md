# 快速开始指南

## 🚀 立即开始开发

### 1️⃣ 安装依赖

打开终端，在项目根目录执行：

```bash
npm install
```

这将安装以下核心依赖：
- **three** - Three.js 3D引擎
- **@tweenjs/tween.js** - 动画补间库
- **vite** - 快速开发服务器

### 2️⃣ 启动开发服务器

```bash
npm run dev
```

成功后会看到：
```
VITE v5.x.x  ready in xxx ms

➜  Local:   http://localhost:5173/
➜  Network: use --host to expose
```

在浏览器中打开 `http://localhost:5173/` 即可看到游戏！

### 3️⃣ 开始开发

现在你已经有了：

✅ **完整的游戏策划案** - `游戏策划案.md`
✅ **开发技能包** - `.codebuddy/skills/web-3d-game/`
✅ **基础项目结构** - 已经搭建好
✅ **示例代码** - `src/main.js` 有基础场景

## 📁 项目结构说明

```
AIGame/
├── src/

│   └── audio/               ← 放置音效文件
└── index.html               ← HTML入口
```

## ❓ 常见问题

### Q: 如何添加恐龙模型？

A: 
1. 下载 GLTF/GLB 格式的恐龙模型（推荐 Sketchfab）
2. 放到 `public/models/` 目录
3. 参考 Skill 中的 "加载GLTF模型" 部分
4. 替换 `main.js` 中的 `createPlayer()` 方法

### Q: 如何实现WASD控制？

A:

### Q: 如何添加进化系统？
