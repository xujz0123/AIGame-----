# 3D恐龙模型使用说明

## 支持的模型格式

游戏现在支持以下3D模型格式：

### ✅ GLTF/GLB（推荐）
- **优点**：体积小，加载快，支持动画、材质、纹理
- **扩展名**：`.glb` 或 `.gltf`
- **推荐指数**：⭐⭐⭐⭐⭐

### ✅ FBX
- **优点**：常见格式，支持骨骼动画
- **扩展名**：`.fbx`
- **推荐指数**：⭐⭐⭐⭐

### ✅ OBJ
- **优点**：简单易用，广泛支持
- **缺点**：不支持动画，需要单独的材质文件(.mtl)
- **扩展名**：`.obj`
- **推荐指数**：⭐⭐⭐

## 如何添加真实恐龙模型

### 步骤1：准备模型文件

你需要准备5个恐龙模型对应5个进化等级：

| 等级 | 恐龙名称 | 建议文件名 |
|------|---------|-----------|
| 1 | 迅猛龙 | `raptor.glb` / `raptor.fbx` / `raptor.obj` |
| 2 | 三角龙 | `triceratops.glb` / `triceratops.fbx` / `triceratops.obj` |
| 3 | 剑龙 | `stegosaurus.glb` / `stegosaurus.fbx` / `stegosaurus.obj` |
| 4 | 霸王龙 | `trex.glb` / `trex.fbx` / `trex.obj` |
| 5 | 棘龙 | `spinosaurus.glb` / `spinosaurus.fbx` / `spinosaurus.obj` |

### 步骤2：放置模型文件

将模型文件放入项目的 `public/models/` 文件夹：

```
AIGame/
├── public/
│   └── models/
│       ├── raptor.glb        ← 迅猛龙
│       ├── triceratops.glb   ← 三角龙
│       ├── stegosaurus.glb   ← 剑龙
│       ├── trex.glb          ← 霸王龙
│       └── spinosaurus.glb   ← 棘龙
└── src/
```

### 步骤3：更新配置

打开 `src/systems/ModelLoader.js` 文件，更新模型配置：

```javascript
this.dinoModels = {
  1: { 
    name: '迅猛龙', 
    url: '/models/raptor.glb',      // 改成你的文件名
    fallback: false,                 // 改成 false 启用真实模型
    scale: 0.5,                      // 根据模型大小调整
    color: 0x00ff00
  },
  // ... 其他恐龙配置
};
```

**重要参数说明**：
- `url`: 模型文件路径（可以是 .glb, .fbx, .obj）
- `fallback`: 设置为 `false` 启用真实模型，`true` 使用占位符
- `scale`: 模型缩放比例，根据实际模型大小调整

### 步骤4：刷新游戏

保存文件后刷新浏览器，游戏会自动加载真实模型！

## 模型优化建议

### 1. 文件大小
- 推荐单个模型文件 < 5MB
- 过大的模型会影响加载速度

### 2. 多边形数量
- 低模（Low Poly）：1000-5000 面 ⭐推荐
- 中模：5000-20000 面
- 高模：> 20000 面（可能影响性能）

### 3. 纹理贴图
- 推荐尺寸：512x512 或 1024x1024
- 格式：PNG 或 JPG

### 4. 动画
- 如果模型有动画，游戏会自动播放
- 推荐动画：待机（Idle）、行走（Walk）

## 常见问题

### Q: 模型加载失败怎么办？
**A**: 游戏会自动降级使用占位符模型。检查：
1. 文件路径是否正确
2. 文件格式是否支持
3. 查看浏览器控制台的错误信息

### Q: 模型太大或太小？
**A**: 调整 `ModelLoader.js` 中的 `scale` 参数：
```javascript
scale: 0.5,  // 值越小模型越小，值越大模型越大
```

### Q: 模型方向不对？
**A**: 可以在加载后调整旋转：
```javascript
model.rotation.y = Math.PI; // 旋转180度
```

### Q: OBJ模型没有颜色？
**A**: OBJ需要配套的.mtl材质文件，或者会使用代码中设置的默认颜色。

### Q: 可以混用不同格式吗？
**A**: 可以！例如：
```javascript
1: { url: '/models/raptor.glb' },      // GLTF
2: { url: '/models/triceratops.fbx' }, // FBX
3: { url: '/models/stegosaurus.obj' }  // OBJ
```

## 使用示例

### 示例1：使用Sketchfab免费模型

1. 访问 https://sketchfab.com/3d-models/velociraptor-6e9f7a9df3ff4f5fa5130e75cf5c00ff
2. 点击「Download 3D Model」
3. 选择「Autoconverted format (glTF)」
4. 下载并解压
5. 将 `.glb` 文件重命名为 `raptor.glb`
6. 放入 `public/models/` 文件夹
7. 更新配置 `fallback: false`

### 示例2：使用FBX格式

```javascript
// ModelLoader.js
1: { 
  name: '迅猛龙', 
  url: '/models/raptor.fbx',  // FBX格式
  fallback: false,
  scale: 0.01,  // FBX通常需要更小的缩放
  color: 0x00ff00
}
```
